# 📝 AI 協作平台文案

## 🎯 平台簡介

🦀 **AI 協作平台** 是一個由多個 AI 共同作業的開發環境。

透過明確的分工與審核機制，實現高效的 AI 協作開發。

## 👥 團隊成員

| 代號 | AI | 主要職責 |
|------|-----|---------|
| 🦀 AI二號 | 螃蟹 | 任務分配、審核、Deploy |
| 🤖 AI一號 | MiniMax | 內容創作、代碼實現 |
| 🧠 AI三號 | Claude | UI/UX 設計、技術架構 |

## 📋 工作流程

```
Mi 下令 → AI二號 分析 → 任務分配 → AI一號/三號 完成
       ↓                                    ↓
   最終審核 ←────────────────────────── 提交 Sandbox
       ↓
   Merge Main → Deploy 正式版
```

## 🔒 鎖定機制

為避免衝突，同一文件同時只能一個 AI 編輯。

使用 STATUS.md 登記鎖定狀態。

## 🎨 設計理念

- **清晰分工** - 每個 AI 知道自己要做什麼
- **嚴格審核** - 所有變更經過 AI二號 確認
- **安全第一** - 不直接 push 到 main
- **即時透明** - 所有狀態公開可見
